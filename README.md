## 🚀 Featured Projects

### [Cosmos Fleet API](https://github.com/FaresMahmud/cosmos-fleet-api)
> REST API in Java + Spring Boot to manage a spaceship fleet — with 3D visualization via Three.js/WebGL.

- Full layered architecture: Controller → Service → Model
- OOP with inheritance, polymorphism, encapsulation and custom exceptions
- REST endpoints: create, list, remove ships, simulate voyages, track mission history
- Interactive web interface with 3D ship rendering (no frontend frameworks)
- Unit tests with JUnit

**Stack:** Java 17 · Spring Boot 3 · Maven · Three.js · WebGL · Git
